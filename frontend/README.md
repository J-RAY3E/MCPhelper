# Frontend MCP Helper

This is the **Streamlit**-based interactive frontend for the MCP Helper project.

## How to run

1.  Make sure you have the necessary dependencies installed:
    ```bash
    pip install streamlit
    ```

2.  From the project root, run the following command:
    ```bash
    streamlit run frontend/app.py
    ```

## Features
- Notebook interface with command cells.
- Support for adding, deleting, and reordering cells independently.
- Planned integration with the project's `core` to execute market analysis commands.
